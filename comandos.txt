# Criar package.json
> npm init

# Instalar dependencias
* express - 
* express-handlebars - Template (imprimir dados do back-end para o front-end)
* boby-parser - Pega requisição para adicionar dados no banco 
* sequelize - Faz abristração para consulta, insert, update, delete...
* sqlite3 - Comunucação com Banco de dados
* 
> npm install express express-handlebars body-parser sequelize sqlite3

# Instalar dependência DEV
* nodemon - Atualiza alterações imediatas sem precisar restart servidor
> npm install -D nodemon

# Inicializar servidor Node com nodemon
> npm run dev
